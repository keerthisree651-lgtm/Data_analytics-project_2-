# Exploratory Data Analysis (EDA) – Project 2

## Overview

This project focuses on **Exploratory Data Analysis (EDA)** using Python to understand the dataset before performing advanced analytics. The objective is to uncover meaningful patterns, identify trends, detect outliers, and summarize important insights through statistical analysis and data visualization.

The project follows a complete EDA workflow including data loading, preprocessing, descriptive statistics, visualization, correlation analysis, and business insights.

This project was completed as part of the **DecodeLabs Data Analytics Industrial Training Program (Batch 2026)**. :contentReference[oaicite:0]{index=0}

## Objectives

- Understand the dataset structure
- Perform descriptive statistical analysis
- Identify missing values and duplicates
- Analyze numerical and categorical features
- Detect outliers using statistical methods
- Explore feature relationships using correlation
- Visualize distributions and trends
- Generate meaningful business insights
